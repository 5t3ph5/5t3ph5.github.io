What is a function?

A function is an action; you can put things inside of it. When you "call it" or "run it" it does the things.

You can define and specify what your function does so that it does specific things (like changing specific traits or characteristics of an element on the page)

Sometimes when we want to run a function on the whole document we say "document.___"
